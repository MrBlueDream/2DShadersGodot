# Cloud Shader for Godot 4

A beautiful, procedurally generated cloud shader for 2D top‑down games.

## Features
- Fully procedural – no external textures needed
- Multi‑layer parallax scrolling
- Pixelation option for retro look
- Color‑tintable clouds
- Breathing pulse animation
- World‑fixed clouds (stay in place as the player moves)

## Quick Start
1. Create a large `Sprite2D` covering your game world.
2. Create a `ShaderMaterial` and assign this shader.
3. Apply the material to the sprite.
